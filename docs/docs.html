quick start

Install the component by inserting the following code in your HTML's <head>: 

<script 
    src="github.com/maoivy/6s063-currency/src/main.js" 
    type="module"
>
</script>

html

Attributes

The component has one reflected attribute-property, expanded, that determines the list of currencies shown. By default, the component displays a short list of the most commonly-used currencies; if expanded is set to true (or "true"), the full list of available currencies is used.

In addition, the component also features a number of non-reflected attributes to allow you to initialize the component: default_input_val, default_input_currency, and default_output_currency.

For example, to initialize the component to convert 5 dollars from USD to CAD, with the full currency list: 

<currency-converter 
    default_input_val="5"
    default_input_currency="USD"
    default_output_currency="CAD"
    expanded="true"
></currency-converter>

Structure

The component has one child, with the following structure, attached to the shadow root: 

<div id="currency-converter">
    <div part="inputs">
        <div part="group">
            <label for="inputVal" part="label">Input Value</label>
            <input type="number" name="inputVal" id="input-val" placeholder="Amount to convert..." part="input-val">
        </div>

        <div part="group">
            <label for="inputCurrency" part="label">Input Currency</label>
            <select name="inputCurrency" id="input-currency" part="currency-select">
                <option value="" disabled selected>From...</option>
            </select>
        </div>
    </div>

    <div part="outputs">
        <div part="group">
            <label for="outputVal" part="label">Output Value</label>
            <div id="output-val" part="output-val"></div>
        </div>

        <div part="group">
            <label for="outputCurrency" part="label">Output Currency</label>
            <select name="outputCurrency" id="output-currency" part="currency-select">
            <option value="" disabled selected>To...</option>
            </select>
        </div>
    </div>

    <button id="convert-button" part="convert-button">Convert</button>
</div>

css

Parts

The currency-converter has many customizable parts:

  • convert-button
  • input-val and output-val
  • inputs and outputs
  • each group consisting of a label and input
  • the label for each input
  • the currency-select dropdown menus

For example, to rearrange the layout of the component, the user can use the inputs, outputs, and groups parts to apply a flexbox to the design.

js

Events

The currency-converter responds to click events on the convert-button button, which triggers the recalculation of the currency conversion with new input values.

Accessors

The following variables can be accessed:

  • expanded (reflected attribute-property)
  • inputVal and outputVal
  • inputCurrency and outputCurrency
