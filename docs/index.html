overview

This component takes in a given value and currency pair and converts the value from one currency to the other.

Install the component by inserting the following code in your HTML's <head>: 

<script 
    src="github.com/maoivy/6s063-currency/src/main.js" 
    type="module"
>
</script>

html

Attributes

The component contains default-input-val, default-input-currency, and default-output-currency attributes to allow you to initialize the component.

For example, to initialize the component to convert 5 dollars from USD to CAD: 

<currency-converter 
    default-input-val="5"
    default-input-currency="USD"
    default-output-currency="CAD"
></currency-converter>

css

Parts

The currency-converter has many customizable parts:

  • convert-button
  • input-val and output-val
  • inputs and outputs (if the user wishes to rearrange the layout of the component)
  • and the label for each option

js

Events

The currency-converter responds to click events on the convert-button button, which triggers the recalculation of the currency conversion with new input values.

Accessors

inputVal, outputVal, inputCurrency, outputCurrency, label
