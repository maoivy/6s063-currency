overview

This component takes in a given value and currency pair and converts the value from one currency to the other.

Install the component by inserting the following code in your HTML's <head>: 

<script 
    src="github.com/maoivy/6s063-currency/src/main.js" 
    type="module"
>
</script>

css

Parts

The currency-converter has three customizable parts: convert-button, value, and label.

js

Events

The currency-converter responds to click events on the convert-button button, at which point it recalculates the currency with the new input values.

Accessors

inputVal, outputVal, inputCurrency, outputCurrency, label
