overview

This component takes in a given value and currency pair and converts the value from one currency to the other.

demos

Converting 5 USD to CAD

<currency-converter 
    default_input_val="5"
    default_input_currency="USD"
    default_output_currency="CAD"
></currency-converter>

Converting 100 JPY to USD

<currency-converter 
    default_input_val="100"
    default_input_currency="JPY"
    default_output_currency="USD"
></currency-converter>

GBP to EUR converter

<currency-converter 
    default_input_currency="GBP"
    default_output_currency="EUR"
></currency-converter>

Unit conversion between any two currencies

<currency-converter 
    default_input_val="1"
    expanded="true"
></currency-converter>

Open-ended (no default specified) converter

<currency-converter
></currency-converter>
